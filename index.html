GCP 架站 - 建立 VM 執行個體 + 設定固定 IP

發表於 2019-11-22 分類於 網站架設 文章字數： 399 所需閱讀時間 ≈ 1 分鐘

僅作為個人紀錄使用

一、建立 VM 執行個體

點選「建立執行個體」

  1. 填寫該執行個體名稱
  2. 根據所在國家及地區選擇適合的區域
  3. 根據網站的所需的規格選擇機器類型，可選擇配好的機器，也可自訂規格，右上角可看到月付的預估值
  4. 配置開機磁碟，點選變更，可選擇該 VM 的作業系統及磁碟類型(永久/SSD)，並配置所需的磁碟空間
  5. 設定防火牆，勾選「允許 HTTP 流量」，若是有網站有 SSL，也一併勾選「允許 HTTPS 流量」
  6. 接著即可按建立，並等待系統建置

SSL 安全憑證設定 (Apache + CentOS 7）

發表於 2019-09-19 分類於 網站架設 文章字數： 487 所需閱讀時間 ≈ 2 分鐘

僅作為個人紀錄使用

1. 取得 SSL 憑證檔案

申請 SSL 憑證後，通常會獲得以下檔案：

  • certificate.crt – 伺服器憑證（主要憑證）
  • private.key – 憑證私鑰（用於驗證伺服器身份，請妥善保管）
  • ca_bundle.crt – 中繼憑證（根憑證與中介憑證的組合，部分 CA 可能不提供）

如果你的 SSL 憑證發行商提供的是 .pem 格式，可能需要轉換或拆分成 .crt 和 .key 以符合 Apache 的需求。
